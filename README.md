# sortingalgs
repo of sorting algs

After compiling your cpp file, run [./{name for compiled code} < data.txt > output.txt] to actually read and write data. 

So for Nbitradix.cpp, you'd run [g++ radix/Nbitradix.cpp -o Nbitradix] to compile it. \
Then run [./Nbitradix < input.txt > output.txt] to let radix.cpp read the input data and write to output.txt.
